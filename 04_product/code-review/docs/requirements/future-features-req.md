# 将来機能リスト

優先度：Low（Future スコープ）

## 機能候補

### 行クリック詳細解説
- コードビューアで行をクリックすると、その行の詳細解説が右側に表示される
- 「この行は何をしているのか？」を即座に確認できる

### 履歴保存
- Railway PostgreSQL を追加してセッション履歴を保存
- 過去の解説をいつでも振り返れる
- テーブル設計: `sessions(id, code, language, level, explanation, created_at)`

### 共有リンク
- 解説結果をURLで共有できる
- 例: `code-review.vercel.app/share/abc123`
- 公開設定（誰でも見れる）と非公開設定

### GitHubリポジトリ全体の構造解説
- リポジトリURLを入力するとファイル構造を読み込み
- 「このリポジトリはどんな構造になっているか」を解説
- agent-server の `list_files` + `read_file` を組み合わせて実現

### 認証機能
- Supabase Auth でログイン
- ユーザーごとの履歴管理

### チーム機能
- チームメンバー間で解説を共有
- コメント機能

### 多言語UI
- 英語インターフェース対応

### VS Code拡張機能
- エディタ上で選択したコードを直接解説
- サイドバーに解説を表示

### コードの変更追跡
- Gitのdiffを解説する機能
- 「このコミットで何が変わったか」をわかりやすく説明
