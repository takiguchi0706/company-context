# 言語自動検出 要件定義書

優先度：Medium

## 概要

GitHub URLやファイルパスからコードを読み込んだ場合、ファイル拡張子から言語を自動検出してセレクトボックスに反映する。

## 検出ルール

| 拡張子 | 言語 |
|--------|------|
| `.ts`, `.tsx` | TypeScript |
| `.js`, `.jsx` | JavaScript |
| `.py` | Python |
| `.go` | Go |
| `.rs` | Rust |
| `.java` | Java |
| `.cpp`, `.cc`, `.cxx` | C++ |
| `.php` | PHP |
| `.rb` | Ruby |
| それ以外 | その他 |

## フォールバック

- 拡張子が不明な場合、またはコード直接貼り付けの場合は「その他」をデフォルト
- ユーザーは常にセレクトボックスで手動変更できる

## 実装場所

`/api/github/route.ts` のレスポンスに `detected_language` フィールドを含める。
クライアント側でセレクトボックスの値を更新する。
