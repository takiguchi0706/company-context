# 解説表示UI 要件定義書

優先度：High

## 概要

解説ページ（`/explain`）の画面レイアウトと表示要件。

## 画面レイアウト

### デスクトップ（lg以上）

```
┌─────────────────────────────────────────────────────┐
│  ヘッダー: code-review  [言語] [レベル] [別のコードで試す]  │
├──────────────────────┬──────────────────────────────┤
│                      │  解説コンテンツ（スクロール可）  │
│  コードビューア       │                              │
│  （左固定スクロール）  │  Q&Aチャット履歴             │
│                      ├──────────────────────────────┤
│                      │  [質問バー] [送信]            │
└──────────────────────┴──────────────────────────────┘
```

### モバイル（lg未満）

```
┌─────────────────────┐
│  コードビューア       │
├─────────────────────┤
│  解説コンテンツ       │
├─────────────────────┤
│  Q&Aチャット履歴     │
├─────────────────────┤
│  [質問バー] [送信]   │
└─────────────────────┘
```

## 左カラム: コードビューア

- `react-syntax-highlighter`の `atomOneDark` テーマ
- 行番号表示（`showLineNumbers={true}`）
- フォント: `JetBrains Mono`
- ヘッダー: 言語名 + レベルバッジ + コピーボタン
- コードが長い場合も独立してスクロール（`overflow-y: auto`）

## 右カラム: 解説エリア

- 上部メタ情報（固定、スクロールしない）:
  - 🕐 実行時間
  - 📊 入力トークン / 出力トークン
  - 💰 コスト（USD）

- 解説コンテンツ（スクロール可）:
  - `react-markdown + remark-gfm` でレンダリング
  - コードブロックは適切にハイライト
  - 見出し・リスト・テーブルをサポート

- チャット履歴（解説の下に積み上がる）:
  - 質問: 右寄せ吹き出し
  - 回答: 左寄せ、Markdownレンダリング

## 右下: 質問バー（固定）

- `<input type="text" placeholder="このコードについて質問する...">`
- 送信ボタン
- ローディング中は入力欄とボタンをdisable
- Enterキーでも送信可能

## アクションボタン

- 「📋 解説をコピー」: クリップボードに解説テキストをコピー
- 「🔄 別のコードで試す」: 入力ページに戻る
