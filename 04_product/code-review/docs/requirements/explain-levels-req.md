# 解説レベル3段階 要件定義書

優先度：High

## 概要

AIの解説レベルを3段階で切り替える。どのレベルでも「コード説明本」スタイルの章構成で解説する。

## 解説の章構成（全レベル共通）

1. **このコードは何をするのか**（全体像・目的）
2. **仕組みの解説**（処理の流れを順に追う）
3. **関数・クラス・変数の関係**（構造・依存関係の説明）
4. **なぜこう書くのか**（設計理由・代替案）
5. **どこに影響するのか**（依存・副作用・呼び出し元）
6. **専門用語の解説**（コード内で使われている技術用語の説明）

## レベル定義

### 🧒 小学生向け

- 身近なたとえ（料理・ゲーム・買い物・おもちゃ）を使う
- 専門用語は使わない、使う場合は必ず説明する
- 絵文字・箇条書きを使って読みやすくする
- 「なぜこうなるの？」を大事に、丁寧に答える

### 🧑‍🎓 中高生向け

- 変数・関数・ループ・条件分岐などの基本用語はOK
- 応用的な概念は噛み砕いて説明する
- たとえは引き続き使うが、より具体的に
- コードの処理の流れを1ステップずつ追う

### 👨‍💻 エンジニア向け

- 専門用語・設計パターン・計算量（Big O）の話をする
- 代替実装との比較
- パフォーマンス・セキュリティ観点でのコメント
- 依存関係・副作用・テスタビリティへの言及
- 実装判断の背景まで含めた深い解説

## UIデザイン

- ラジオボタン形式（3択）
- 選択中のレベルをハイライト
- 解説ページに「レベル: 小学生向け」バッジを表示
