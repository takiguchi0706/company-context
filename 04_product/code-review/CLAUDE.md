# CLAUDE.md — code-review

## プロダクト概要

**書いたコードを、コード説明本のように解説してくれるツール。**
コードを貼り付けるか、GitHubのURLまたはリポジトリパスを指定するだけで、AIが小学生から
エンジニアまで3段階のレベルで「なぜこのコードがこう動くのか」を丁寧に解説する。
解説中の疑問はその場でチャット質問できる。

---

## ビジョン

「コードを書いた人が、それがなぜ動くのか、他人（小学生レベルから）に伝えられる状態にするツール」

- コードを「おまじない」から「理解できるもの」に変える
- 解説は「コード説明本」スタイル——全体像 → 仕組み → 関数・クラスの関係 → 設計理由 → 影響範囲 → 専門用語
- デバッグを一緒にするような感覚で、なぜ動くかを一歩ずつ追う

---

## 技術スタック

- **フロントエンド**: Next.js 16 + TypeScript + Tailwind CSS v4
- **UIライブラリ**: react-syntax-highlighter（コード表示）、react-markdown + remark-gfm（解説表示）、lucide-react（アイコン）
- **AI連携**: agent-server（Railway稼働中）経由でClaude API呼び出し
- **GitHub連携**: GitHub REST API（プライベートリポジトリはGITHUB_TOKEN PAT経由）
- **デプロイ**: Vercel

---

## 対象ユーザー

- 自分のコードの理解を深めたい個人開発者
- コード教育に使いたい人（学習者・教育者）
- コードレビューの観点を学びたいエンジニア

---

## 設計方針

- **ステートレス**：DB無し、認証無し。sessionStorageで一時保持（ページリロードで消える）
- **シンプル・高速**：1ファイル貼り付け → 即解説。余計な登録不要
- **モバイル対応**：縦並び / デスクトップ = 左右2カラム
- **将来的に Railway PostgreSQL で履歴保存予定**

---

## 画面構成

1. **入力ページ** (`/`)
   - Tab A: コード直接貼り付け
   - Tab B: GitHub URL入力（`github.com/owner/repo/blob/branch/path`）
   - Tab C: 自社リポジトリパス入力
   - 言語選択 / 解説レベル選択 / モード（解説 or レビュー）

2. **解説ページ** (`/explain`)
   - 左: シンタックスハイライト付きコードビューア
   - 右上: AI解説（Markdown）
   - 右下: 質問バー（追加質問 → チャット形式で回答が追加）

## 環境変数

| 変数名 | 説明 | 必須 |
|--------|------|------|
| `AGENT_SERVER_URL` | agent-server のURL | ✅ |
| `GITHUB_TOKEN` | プライベートリポジトリアクセス用 PAT | オプション（未設定時はパブリックのみ） |
